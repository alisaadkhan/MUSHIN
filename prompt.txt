2. **Mandatory Metrics**:
   - Followers/Subscribers are NO LONGER OPTIONAL. 
   - Every search card MUST display a follower/subscriber count.

   - Display label MUST be "Subscribers" for YouTube creators.
   - Profile images MUST work (Fix the issue where some creators show placeholder icons).

   - Tighten `snippetRelevanceScore` and `inferNiche` to reject high-confidence distractors.
